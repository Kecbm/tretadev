<h2 id="execute">Execute the project 💻</h2>

<br>

<details><summary>React Application 🎉</summary>
<p>
To install the dependencies and start the React application, run the commands in the following order:

```bash
  cd tretadev
```

```bash
  cd src
```

```bash
  npm install
```

```bash
  npm start
```

</p>
</details>
