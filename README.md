<h2 id="execute">Executar o projeto 💻</h2>

<br>

<details><summary>Aplicação React 🎉</summary>
<p>
Para instalar as dependências e iniciar a aplicação React, execute os comandos na ordem a seguir:

```bash
  cd tretadev
```

```bash
  cd src
```

```bash
  npm install
```

```bash
  npm start
```

</p>
</details>
